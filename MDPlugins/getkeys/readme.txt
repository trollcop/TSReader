Emunation softcam.key creator 1.0.0 ( 01/31/07 )

How to use

1. These files must be in the same folder
	.\getKeys.exe
	.\getKeys.ini

2. getKeys.ini must include all parameters
	-filepath
	-viewResult
	-website
	-spaced
	-dishKeyTag0,dishKeyTag1
	-bevKeyTag0,bevKeyTag1
   See the getKeys.ini files for parameters description

3. In getKeys.ini, line starting with '#' are commented

4. Parameter value MUST be surrounded by quotes (ex.: "value")

5. Any website which contain keys can be used:
	-enter the complete adress without "http://"
	-indicated if there is spaces
        -enter the tag before each keys
	


FILEPATH examples
	filepath="C:\AltDvb\Plugins\Av0"
	filepath="C:\Program Files\mytheatre\plugins"
	filepath="." ( current folder )

NOTES
	- The filepath between the quotes must exist or an error
          will occur.
